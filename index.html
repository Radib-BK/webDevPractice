Now For some Notes on Css :


  1. Use 'Color Hunt' to see color pallets with hex codes "COLOR HUNT"

  3. Know about the css default attribute values from CSS-Default
  4. Details of all the attributes in CSS-REFERENCE
  5. For selecting a specific thing to apply css, we need to set a 'CLASS' attribute in that tag. and then use '.ClassName' to access.
  6. Or, for selecting a specific thing to apply css, we need to set a 'ID' attribute in that tag. and then use '#idName' to access. The difference between id and class is ID is unique where there can be multiple of same Class. ID attribute can't have more than on element ID="hello pizza" but CLASS attribute can have more than one element CLASS="hello pizza".
